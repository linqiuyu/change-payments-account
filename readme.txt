Change woocommerce-payments account.

== Description ==

You can only connect one WooCommerce Payments account per site. This plugin allows you to quickly switch WooCommerce Payments accounts when you need to change your domain name.

每个站点只能授权一个 WooCommerce Payments 账号，当你需要更改域名时，这个插件可以帮你快速切换 WooCommerce Payments 账号。

== Frequently Asked Questions ==

= invalid connect =

When you use the current domain name connect WooCommerce Payments, the previous connect will be invalidate.
